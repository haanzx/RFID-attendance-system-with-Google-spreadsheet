# RFID-attendance-system-with-Google-spreadsheet
RFID attendance system using ESP32 as a microcontroller and Google spreadsheet as a database and data summary

RFID RC-522
LCD I2C 16x2

PINOUT
RFID    ESP32
SDA  =>  D5
SCK  =>  D18
MOSI =>  D23
MISO =>  D19
IRQ  =>  NC
GND  =>  GND
RST  =>  D4
3.3V =>  3V3

LCD   ESP32
GND => GND
VCC => VIN
SDA => D21
SCL => D22
